UI Redesign Scope

The CivicPulse inspiration has a lot of design ideas. How much do you want to adopt?

Key differences from our current UI

Current (Berlin Leben)

  • Dark theme (#0f0f13)
  • Geist Sans / Mono fonts
  • Indigo primary (#6366f1)
  • Simple cards, rounded-xl
  • Emoji icons
  • Compact navbar, max-w-5xl
  • Minimal stat cards

Inspiration (CivicPulse)

  • Light theme (#f7f7f3)
  • Space Grotesk + Manrope fonts
  • Teal primary (#1b6565), gold + cyan accents
  • Bento grid, rounded-[2rem]
  • Material Symbols icons
  • Sidebar nav + bigger layout
  • Circular gauges, bold stat cards
A

Full Redesign

Switch to light theme, new color palette, new fonts, bento grid dashboard, sidebar nav, Material Symbols, larger border radii, glass effects. Basically adopt the CivicPulse aesthetic wholesale.

B

Selective Adoption

Keep dark theme but upgrade specific elements: bolder typography, bento grid dashboard, circular LiD gauge, better stat cards, Material Symbols icons, larger card radii. Cherry-pick the best ideas.

C

Dashboard Focus

Only redesign the dashboard/home page with inspiration from the bento grid layout, stat cards, and visual hierarchy. Leave other pages and the overall theme untouched.
