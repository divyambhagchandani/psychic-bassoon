What kind of tabs do you want in the chat?

The tutor chat currently has a single conversation view. Pick the tab structure that fits your vision.

A

Topic Tabs

Tabs that switch between different tutor "modes" — e.g. Chat (free conversation), Grammar (grammar explanations), Vocabulary (word lookup / flashcards). Each tab has its own conversation context.

B

Conversation Tabs

Multiple independent chat threads — like browser tabs. Start a new conversation about a different topic without losing the old one. Each tab is a separate chat history.

C

Content Tabs (Chat + Tools)

Tabs that mix chat with utility panels — e.g. Tutor (the chat), Word Breakdown (select text → see word-by-word + grammar), Saved Words (your vocabulary list). Different features, not just different conversations.
